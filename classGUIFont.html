Detailed Description

Some notes on markup characters supported by GUIFont functions.

  • \n is a regular line feed.
  • \tXXX, where XXX is decimal offset in pixels. Moves the caret to a position offset by XXX to the right from the start of the line. This is how aligned tables are implemented.
  • \rXXX, where XXX is a decimal offset in pixels. Moves the caret so that the rest of the line is right-justified, with the given offset from the right window border. This is used e.g. in the create party screen.
  • \fXXXXX, where XXXXX is a decimal color code for 16-bit color to use for the text that follows.
See also
Color::fromC16
