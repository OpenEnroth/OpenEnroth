◆ OS_casepath()

std::filesystem::path OS_casepath ( std::filesystem::path  path)

This function emulates windows behavior on posix. You pass in a path, this function traverses it as if the underlying filesystem was case-insensitive, and returns a case-corrected path that actually exists.

On windows this function just returns the path as is.
