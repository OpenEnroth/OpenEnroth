◆ appendTo()

template<class T >
auto appendTo ( std::vector< T > *  dst)

Creates a deserialization wrapper that instructs the binary serialization framework to append deserialized elements into the target vector instead of replacing its contents. This is useful when you want to deserialize several sequences into the same std::vector object - normally subsequent deserialize calls would replace the vector contents, but using this wrapper lets you accumulate the results of several deserialize calls instead.

Parameters
dstVector to append deserialized elements to.
Returns
Wrapper object to be passed into deserialize call.
