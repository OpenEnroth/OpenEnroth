Detailed Description

An Overlay is a screen built in Immediate Mode GUI ( nuklear ) which is displayed on top of any other ui and it's mainly used for debug purposes ( Console, Render Stats, Debug View Buttons, etc... ).

The Overlay System is the class responsible to keep track of all the registered Overlays and update them accordingly.
